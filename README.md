# Tbnguyen1407 Tap

## How do I install these formulae?

`brew install tbnguyen1407/tap/<formula>`

Or `brew tap tbnguyen1407/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tbnguyen1407/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
